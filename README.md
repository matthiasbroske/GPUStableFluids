## System Requirements
- Unity 2022.3 or later

## Project Layout
The project consists of two demo scenes in the `Assets/Scenes` folder:
### `2D Stable Fluids`
- **Left Click**: Stir fluid
- **Right Click**: Add dye
- **R**: Reset simulation
### `3D Stable Fluids`
- **Left Click**: Rotate
- **Right Click**: Zoom
- **R**: Reset simulation
